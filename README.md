## 🚀 Featured Project

**[Sous Chef](https://github.com/akshaya-01012000/sous-chef)** — AI-powered meal planner & recipe generator

A full-stack **Progressive Web App** installable on iOS and Android — no app store required.

- 🍽 AI meal suggestions (breakfast, lunch, dinner) based on live pantry inventory
- 📋 Full step-by-step recipe generation per meal with ingredient matching
- 🥗 Per-meal nutrition info — calories, protein, carbs, fat per serving
- 🔍 Pantry search and category filtering
- 🧾 Receipt photo scanner — AI extracts grocery items and adds to pantry
- 🛒 Auto-populated shopping list from missing ingredients
- 📵 Offline support via Service Worker · Dark mode · Installable PWA

Built with: Vanilla JS · HTML5 · CSS3 · Anthropic Claude API · Web App Manifest
